## Description

### Groups

#### Group Targets

One of the following:

| Target | Description |
| --- | --- |
| `Parameter` | Group targets parameter. The group IDs then are parameter IDs. |

##### Parameter Target Names

One of the following:

| Id | Description |
| --- | --- |
| `EyeBlink` | Eye blink parameters group. |
| `LipSync` | Mouth opening parameters group. |
